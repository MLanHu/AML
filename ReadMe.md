## Introduction
This is a pytorch implementation of binary matching case described in *'Adversarial-Metric Learning for Audio-Visual Cross-Modal Matching'*
  
## Usage
* python = 3.6
* torch = 1.4.0+cu100
* librosa = 0.7.2
* torchvision = 0.5.0+cu100
* numpy = 1.19.0
## Run the demo
cd AML
python main.py
